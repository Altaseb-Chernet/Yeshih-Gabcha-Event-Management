# Backend - Yeshih-Gabcha Event Management (In Progress)

This folder contains the backend of the **Yeshih-Gabcha Event Management System**, currently under development using PHP and MySQL.

---

## Overview

The backend will handle:

* User authentication (login & registration)
* Event management (create, update, delete)
* Booking system
* Database operations

---

## Technologies Used

* PHP
* MySQL
* Apache (XAMPP)

---

## Setup (Development)

1. Start Apache and MySQL (XAMPP)

2. Place the project in:

```bash id="b5c2fr"
htdocs/
```

3. Create a database in phpMyAdmin
   (e.g., `yeshih_gabcha`)

4. Configure database connection in your PHP files
