# Frontend - Yeshih-Gabcha Event Management

This folder contains the frontend of the **Yeshih-Gabcha Event Management System**. It is responsible for the user interface and interaction of the application.

---

## Overview

The frontend allows users to:

* View and explore events
* Register and log in
* Book and manage events
* Interact with the system بسهولة

---

## Technologies Used

* HTML & CSS
* JavaScript
* React.js



---

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Start the application:

```bash
npm start
```
